Introducere

Romanian traffic signs tutor este un website destinat învățării legislației și a semnelor rutiere din România. Pe lângă acestea, utilizatorului îi sunt disponibile semnele rutiere din 5 alte țări: Anglia, Franța, Italia, Turcia și Islanda

Tehnologii utilizate

Pentru realizarea proiectului au fost utilizate SCSS și JavaScript pentru realizarea design-ului responsiv. Proiectul a fost realizat conform metodologiei BEM. Majoritatea imaginilor utilizate sunt în format vectorizat pentru a asigura păstrarea calității.

Paginile cu conținut masiv(paginile pentru semnele rutiere) au fost generate prin utilizarea unui scrip de Python care a extras informațiile de pe site-urile oficiale într-un JSON, iar apoi acestea au fost transformate în HTML-uri prin folosirea unui program Java.
